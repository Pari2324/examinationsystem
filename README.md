# Secure Online Examination System

Week 1: Static Responsive Landing Page

Developed a modern dark-themed responsive landing page using HTML5 and CSS as part of Web Technology PBL.

Includes:
- Header
- Hero Section
- About System
- Features
- User Roles
- Call to Action
- Footer

## Week 2: Interactive JavaScript Elements

Enhanced the static UI by adding interactivity using JavaScript and DOM manipulation.

Features added:
- Theme Toggle (Dark/Light Mode)
- Mobile Navigation Menu
- Smooth Scrolling
- Scroll to Top Button
- Dynamic Feature Loading
- Animated Stats Section
- Toast Notification

---

## Project Objective

To build a secure and responsive online examination system step-by-step using modern web technologies.
